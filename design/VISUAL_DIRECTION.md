# Personal Website – Visual Direction

# 1. Vision

The website should feel like the personal website of someone who builds world-class digital products.

It should not resemble a resume.

It should not resemble a SaaS landing page.

It should feel timeless.

Every visual decision should communicate craftsmanship.

---

# 2. First Impression

Within the first three seconds, visitors should feel:

- Premium
- Calm
- Modern
- Intentional

The design should immediately communicate that significant attention has been given to detail.

---

# 3. Visual Inspiration

The quality benchmark is inspired by products such as:

- Linear
- Raycast
- Vercel
- Stripe
- Notion
- Apple

Do not imitate their layouts.

Instead, match their level of refinement, consistency, and attention to detail.

---

# 4. Overall Style

Minimal.

Large whitespace.

Strong typography.

Few visual elements.

Every element should have a purpose.

---

# 5. Layout Philosophy

One long scrolling experience.

Clear visual rhythm.

Alternating content density.

Generous breathing room between sections.

Readable line lengths.

No unnecessary columns.

---

# 6. Background

The background should never feel empty.

Instead:

Subtle gradients.

Very soft lighting.

Extremely light texture.

No heavy patterns.

No visual noise.

---

# 7. Cards

Cards should feel lightweight.

Rounded corners.

Minimal borders.

Soft shadows.

Large internal spacing.

Avoid glassmorphism everywhere.

Use glass only where it has purpose.

---

# 8. Typography

Typography is the hero.

Large headlines.

Comfortable body text.

Strong hierarchy.

Excellent readability.

Very few font weights.

Consistent spacing.

---

# 9. Images

Prioritize:

Product screenshots.

Device mockups.

Clean diagrams.

Avoid:

Stock photography.

Decorative illustrations.

Random 3D graphics.

---

# 10. Color Usage

Color communicates hierarchy.

Accent colors should be rare.

Most of the interface should rely on spacing and typography rather than color.

---

# 11. Hero

The hero should occupy nearly the full first viewport.

Large statement.

Minimal supporting text.

One primary CTA.

One secondary CTA.

Immediate visual focus on what I'm building.

---

# 12. Section Rhythm

Alternate between:

Text-heavy.

Visual-heavy.

Story.

Evidence.

Project.

Reflection.

The website should feel like reading a carefully written article rather than browsing a dashboard.

---

# 13. Motion Integration

Motion should reinforce structure.

Sections should reveal naturally.

Cards should appear with subtle transitions.

Scrolling should feel effortless.

Motion should never become the main attraction.

---

# 14. Mobile

The mobile experience should feel designed rather than compressed.

Typography remains the primary design element.

Sections should stack naturally.

Touch targets remain generous.

---

# 15. Long-Term Scalability

The visual language should support:

More products.

Case studies.

Writing.

Speaking.

Open-source work.

Without redesigning the entire website.
