# Personal Website – Motion System

# Motion Philosophy

Motion exists to improve understanding, reinforce hierarchy, and communicate quality.

Every animation must have a purpose.

Motion should never distract from content.

---

# Motion Principles

- Fast but never abrupt
- Smooth but never slow
- Intentional
- Calm
- Consistent
- Interruptible

---

# Motion Language

The website should feel:

- Refined
- Elegant
- Responsive
- Lightweight

Never:

- Playful
- Chaotic
- Flashy
- Over-animated

---

# Page Entry

The first impression should feel immediate.

The Hero should fade into view with subtle upward movement.

Supporting content should follow with a slight stagger.

---

# Scroll Motion

Sections reveal naturally.

Movement should be minimal.

Scrolling should never trigger excessive animations.

Content remains the hero.

---

# Component Motion

Buttons

- Soft hover transition
- Slight elevation
- No bouncing

Cards

- Gentle lift
- Soft shadow transition
- Minimal scale

Navigation

- Smooth active state transitions
- Underline or indicator movement
- No flashy effects

---

# Timing Philosophy

Micro interactions:
100–200 ms

Standard transitions:
200–350 ms

Large section transitions:
350–500 ms

Avoid long animations.

---

# Easing

Prefer:

Ease Out

Ease In-Out

Avoid:

Bounce

Elastic

Overshoot

Spring-heavy motion

---

# Reduced Motion

Support prefers-reduced-motion.

Replace movement with opacity where appropriate.

Never remove functionality.

---

# Performance

Animations must maintain 60 FPS.

Avoid layout-triggering animations.

Prefer transforms and opacity.

---

# Success Criteria

Visitors should notice that the website feels polished.

They should never notice the animations themselves.
