# Personal Website – Design System

# 1. Design Philosophy

The website should communicate craftsmanship.

Every visual decision should reinforce quality rather than decoration.

The design should age well.

Avoid trends that will look outdated in two years.

---

# 2. Visual Identity

Keywords

- Premium
- Calm
- Minimal
- Intentional
- Technical
- Elegant
- Confident

Not:

- Loud
- Playful
- Corporate
- Flashy
- Futuristic
- Neon

---

# 3. Color Philosophy

The palette should be neutral.

Accent colors should be used sparingly.

Whitespace should create hierarchy instead of heavy borders.

The design should rely on typography and spacing more than color.

---

# 4. Theme

Dark-first.

Light mode may be added later.

Dark mode should feel comfortable during long reading sessions.

Never use pure black.

---

# 5. Typography

Typography is the primary design element.

Large headlines.

Comfortable reading width.

Generous spacing.

Strong hierarchy.

Minimal font weights.

Maximum consistency.

---

# 6. Layout

Large whitespace.

Consistent grid.

Generous margins.

Balanced composition.

Readable content width.

Every section should breathe.

---

# 7. Components

The website should have a very small component library.

Primary Button

Secondary Button

Navigation

Section Container

Project Card

Timeline Item

Tag

Footer

Everything else should be composed from these.

---

# 8. Motion

Motion communicates hierarchy.

Animation should never exist for decoration.

Preferred motion:

Fade

Slide

Scale (minimal)

Opacity

Avoid:

Bounce

Elastic

Overshoot

Continuous floating

Spinning

---

# 9. Imagery

Large screenshots.

High-quality mockups.

Minimal illustrations.

No stock photography.

No meaningless decorative graphics.

---

# 10. Icons

Use a single icon library.

Icons support content.

Never dominate content.

---

# 11. Spacing

Whitespace is a design tool.

Sections should feel open.

Avoid cramped layouts.

Consistent spacing scale throughout the site.

---

# 12. Responsiveness

Desktop

Tablet

Mobile

All three should be intentionally designed.

Not merely resized.

---

# 13. Accessibility

High contrast.

Keyboard navigation.

Semantic HTML.

Reduced motion.

Readable font sizes.

Visible focus states.

Accessible color combinations.

---

# 14. Performance

Animations should not reduce responsiveness.

Images should be optimized.

Fonts should be loaded efficiently.

Target excellent Lighthouse scores.

Minimal layout shift.

---

# 15. Long-Term Evolution

The design system should allow:

Future writing section

Case studies

Talks

Open-source work

New flagship products

Without redesigning the entire website.
