# Personal Website – Information Architecture

# Purpose

This document defines how information is organized across the website.

The objective is to create a reading experience that naturally builds trust, demonstrates capability, and encourages meaningful conversations.

The structure should guide visitors through a deliberate narrative rather than simply presenting sections.

---

# Information Flow

The website follows a single continuous narrative.

Visitors should naturally progress through the following questions:

Who is this person?

↓

What is he building right now?

↓

Can he actually build products?

↓

How does he think?

↓

What experience supports this?

↓

What skills enable this?

↓

How can I work with him?

Every section exists to answer exactly one of these questions.

---

# Site Architecture

Single-page application.

Future expansion may introduce dedicated pages for:

- Case Studies
- Writing
- Open Source
- Speaking

The landing page should remain the primary entry point.

# Mobile Information Hierarchy

The order remains identical.

No content should disappear on mobile.

Instead

Layouts adapt.

Typography scales.

Spacing compresses.

Navigation simplifies.

---

# Scalability

The architecture should support adding

New flagship products

Case studies

Technical writing

Open-source projects

Conference talks

Without restructuring the website.

---

# Information Architecture Principles

Show products before skills.

Show thinking before technologies.

Show evidence before claims.

Guide visitors through a story.

Maintain a clear visual hierarchy.

Reduce cognitive load.

Preserve generous whitespace.

Always optimize for trust.

---

# Success Criteria

A founder, recruiter, or hiring manager should leave the website with the following understanding:

- I build products, not just code.
- I think strategically before implementing.
- I care deeply about quality.
- I can take ownership from concept to execution.
- I would be a valuable addition to a high-performing team.
