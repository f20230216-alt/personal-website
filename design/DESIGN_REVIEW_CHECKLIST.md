# Personal Website – Design Review Checklist

Every screen, section, and component must satisfy this checklist before being considered complete.

---

# First Impression

- Can a first-time visitor understand who I am within 5–10 seconds?
- Is there one obvious focal point?
- Does the page immediately communicate quality?
- Does it avoid looking like a generic developer portfolio?

---

# Visual Hierarchy

- Is the most important information immediately visible?
- Does typography establish a clear reading order?
- Is whitespace being used intentionally?
- Are sections visually distinct without excessive decoration?

---

# Layout

- Does the layout feel balanced?
- Is content aligned consistently?
- Are reading widths comfortable?
- Does every section have enough breathing room?

---

# Content

- Does every section answer a real question a founder or hiring manager would ask?
- Does every sentence add value?
- Are there unnecessary buzzwords?
- Is evidence shown instead of claimed?

---

# Components

- Are components visually consistent?
- Are spacing and sizing consistent?
- Are corner radii consistent?
- Are shadows and borders restrained?

---

# Motion

- Does every animation have a purpose?
- Does motion improve understanding?
- Are animations subtle?
- Does reduced-motion mode remain usable?

---

# Accessibility

- Keyboard accessible
- Proper semantic structure
- High color contrast
- Visible focus states
- Screen-reader friendly
- Responsive typography

---

# Mobile

- Designed intentionally for mobile
- Comfortable touch targets
- Readable typography
- No horizontal scrolling
- Consistent spacing

---

# Performance

- No layout shifts
- Optimized images
- Fast initial render
- Minimal JavaScript
- Smooth scrolling
- Responsive interactions

---

# Branding

- Does the section reinforce the Product Builder identity?
- Does it communicate craftsmanship?
- Does it feel premium?
- Does it feel authentic?

---

# Final Question

Before approving any design, ask:

"If this were the only page a startup founder saw about me, would I confidently expect them to schedule an interview?"

If the answer is not an immediate "Yes," iterate further.
